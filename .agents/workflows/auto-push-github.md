// turbo-all

1. Check repository status
   `git status`

2. Stage all modifications in the current directory only
   `git add .`

3. Review the staged changes to understand what was modified
   `git diff --staged`

4. Analyze the staged changes to automatically generate a clear, concise, and conventional commit message. NEVER ask the user to confirm the message, just commit the code directly!
   `git commit -m "<auto-generated message>"`

5. Push the committed changes to your remote repository on the CURRENT branch
   `git push origin $(git branch --show-current)`
