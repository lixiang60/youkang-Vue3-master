// turbo-all

1. Run code validation (Lint & Typecheck)
   `yarn build:prod`

2. Check repository status
   `git status`

3. Stage all modifications in the current directory only
   `git add .`

4. Review the staged changes to understand what was modified
   `git diff --staged`

5. Analyze the staged changes to automatically generate a clear, concise, and conventional commit message. NEVER ask the user to confirm the message, just commit the code directly!
   `git commit -m "<auto-generated message>"`

6. Push the committed changes to your remote repository on the CURRENT branch
   `git push origin $(git branch --show-current)`
